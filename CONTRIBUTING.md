# Contributing to Unmemorable Team Projects

First of all — thank you for your interest.

We appreciate help from the community, but please read this guide before contributing.

---

## 🧠 Our philosophy

- We make psychological horror mods. No jumpscares.
- Lore comes first. Every mechanic has a reason.
- We value atmosphere over quantity.

---

## 🐛 Reporting bugs

If you find a bug, please report it via:

- **Telegram:** [@kasper31](https://t.me/kasper31)
- **Modrinth:** [kasperoid](https://modrinth.com/user/kasperoid)

**Please include:**
- Mod version
- Minecraft version and loader (Fabric)
- Steps to reproduce the bug
- Crash report (if any)
- Screenshot or video (if possible)

---

## 💡 Suggesting ideas

We welcome ideas, but please:

- Read the existing lore and mechanics first
- Explain *why* your idea fits the mod
- Be specific. "Add more horror" is not specific.

Send ideas via Telegram.

---

## 🎵 Sounds and assets

If you want to contribute sounds or textures:

- Sounds must be **mono .ogg**, CC0 / Public Domain
- Textures must be **pixel art**, Minecraft style
- Send samples before full work

---

## 🚫 What we don't accept

- Pull requests (code is closed source — ARR license)
- Jumpscare suggestions
- Features without lore explanation
- Random "scary monsters"
