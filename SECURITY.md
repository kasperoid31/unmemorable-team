# Security Policy

## Supported Versions

Only the latest stable version of Memory Leak receives security updates.

| Version | Supported |
|---------|-----------|
| 1.0.x (latest) | ✅ Yes |
| 0.5.x and older | ❌ No |

## Reporting a Vulnerability

If you discover a security vulnerability in Memory Leak (or any other Unmemorable Team project), please **do not** report it via public Discord or Telegram channels.

Instead, contact us privately:

- **Telegram:** [@kasper31](https://t.me/kasper31)
- **Discord:** `kasperoid` (mention in the server, then DM)

### What to include

- Clear description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact
- Any suggestions for a fix

### What to expect

- You will receive a response within 48 hours
- We will confirm the issue or ask for more details
- Once fixed, we will credit you in the changelog (if you wish)

## Scope

This policy applies to:

- The Memory Leak mod code
- Official Unmemorable Team tools and scripts

Third-party mods or modified versions are not covered.

## Responsible Disclosure

We ask that you give us reasonable time to fix the issue before disclosing it publicly.

**Thanks for helping keep Memory Leak and the community safe.**
