# 🧠 MEMORY LEAK

> *Your world is not a world. It's a file that was never closed.*

You enter a new world. Everything is as usual.

The sun rises over the horizon. Trees reach for the sky. Caves are dark and damp. You gather wood, craft an axe, build your first house. Torches light up the walls. You feel safe.

But an hour passes. Then another. Then another.

You start noticing strange things. At first, you don't believe yourself. You're just tired, you think.

But no.

You walk along a path you never made. You hear footsteps behind you — you turn around, no one is there. You keep walking — the footsteps return. As if someone is repeating your path. As if someone was here before you. As if someone stayed.

Strange sounds come from nowhere. A whisper you can't understand. Breathing right into your ear. The sound of a closing door in a house that has no doors. You freeze. Silence. Nothing. You start moving again — and it resumes.

The world isn't trying to scare you. There are no monsters jumping out of the dark. No screams, no blood. Just silence. But the silence grows too loud.

You notice that some blocks look different. As if they are old. As if someone touched them before you. As if they remember someone else's hands. You can't explain it — but you feel it.

Sometimes the world acts strangely. The interface flickers. Sounds overlap. For a second, you think the game has broken. But no. It's just... working differently.

You keep playing. Keep building. Keep exploring caves. But the feeling doesn't go away. It's always there. At the edge of your perception. Around the corner. Behind you.

You are not the first one here. And you are not the last.

Memory Leak — a psychological horror mod for Minecraft. No jump scares. No bosses. Just the slow realization that your save file is not entirely yours.

---

## Requirements

- Minecraft 26.1.x
- Fabric + Fabric API
- Java 25

---

## Installation

1. Download the `.jar` file
2. Place it in the `mods` folder
3. Launch the game

---

## Difficulty modes

- **Spooky** — atmosphere, mild dread
- **Normal** — intended balance
- **Hard** — faster and meaner
- **Hardcore** — ruthless
